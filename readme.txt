== Description ==

Add an appropriates arabic fonts from 50 fonts to show your arabic website much better.
Cairo, Lateef, Tajawal, Amiri, Rubik, Sky... and others beautiful fonts.

Key Features:
* Beautiful arabic fonts.
* Select custom font for each element: body, h1, h2, h3, h4, h5, h6.
* Translation ready of the plugin strings to arabic see language folder from the plugin root folder.

This plugin helped website owners who have an Arabic language website

== Installation ==

1. Upload the `khattat-arabic-fonts` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to "Appearrance=> Customize => Khattat Arabic Fonts Settings" then work on the fonts.
